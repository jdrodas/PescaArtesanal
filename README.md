# PescaArtesanal
Aplicaciones usando herramientas MS .NET para demostrar conectividad a bases de datos SQLite, PostgreSQL y MongoDB


## API
### [PescaArtesanalAPI](https://github.com/jdrodas/PescaArtesanal/tree/main/PescaArtesanalAPI)
- WebAPI en .NET 7.x con capa de persistencia de datos en MongoDB 6.x


## Aplicaciones de Escritorio

### [PescaArtesanal_WindowsForms](https://github.com/jdrodas/PescaArtesanal/tree/main/PescaArtesanal_WindowsForms)
- Aplicación en Windows Forms para .NET 7.x para demostrar conectividad a SQLite

### [PescaArtesanal_NoSQL_WindowsForms](https://github.com/jdrodas/PescaArtesanal/tree/main/PescaArtesanal_NoSQL_WindowsForms)
- Aplicación en Windows Forms para .NET 7.x para demostrar conectividad en MongoDB 6.x

## PoC - Pruebas de Concepto

### [PescaArtesanal_PoC_Console](https://github.com/jdrodas/PescaArtesanal/tree/main/PescaArtesanal_PoC_Console)
- Aplicación de Consola en C# para .NET 7.x para demostrar acciones básicas de CRUD sobre una DB SQLite

### [PescaArtesanal_PoC_NoSQL_Console](https://github.com/jdrodas/PescaArtesanal/tree/main/PescaArtesanal_PoC_NoSQL_Console)
- Aplicación de Consola en C# para .NET 7.x  para demostrar acciones básicas de CRUD sobre una DB MongoDB en LocalHost
